## אם משהו נכשל

1. **Settings → System → Logs** → חפש `jane` או `config_api`
2. בעיות נפוצות:
   - **שגיאת auth** → `_get_api_token` נכשל — בדוק parameter של token_type
   - **404 בקריאת API** → entity_id לא הומר — בדוק `resolve_config_id`
   - **Connection refused** → פורט שגוי — בדוק `hass.http.server_port`
3. שלח את שורות הלוג הרלוונטיות לדיבוג
